# Vue 2.7 + Vite3.0

This is a reproduction of memory leak using vite-plugin-checker + vueTsc 


Getting Start

```shell
pnpm install
pnpm run dev
```
Open the *src/main.js*, hold down ctrl+s and don't let go, a memory leak will occur after about 3 minutes
